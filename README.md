# 如何把你的App加入白名单

国内各大厂商，为了优化电量，屏幕关闭以后将会把后台运行的服务关闭，如果你的App需要后台运行或则需要接收推送，那么被系统优化后，你得App将会运行异常，下面整理了各大手机，如何把你的App加入白名单，以免被系统杀掉。
